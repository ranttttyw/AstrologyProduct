
## 技术栈

| 模块 | 技术 | 说明 |
|------|------|------|
| 前端 | React 18 + Babel Standalone | CDN加载，单HTML文件，无构建步骤 |
| 后端 | Python FastAPI + Uvicorn | 异步Web框架 |
| AI | DeepSeek API | temperature=0.95, frequency/presence_penalty |
| RAG | jieba + scikit-learn TF-IDF | 星座知识语义检索，ChromaDB作为可选后端 |
| 星历 | Kerykeion / 纯数学Fallback | 行星位置和相位计算 |
| 数据库 | SQLite | 零配置，单文件 |
| 认证 | JWT + UUID访客登录 | 无需注册，自动生成身份 |
| 部署 | Render (Free Tier) | 自动从GitHub部署 |

## 本地开发

```bash
# 1. 进入后端目录
cd astrology-bot

# 2. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
echo "DEEPSEEK_API_KEY=你的key" > .env

# 5. 启动（同时serve前端和API）
python -m uvicorn main:app --port 8000

# 6. 打开浏览器
# http://localhost:8000
```

## 部署到Render

1. 推到GitHub仓库
2. 在 render.com 创建Web Service，连接GitHub仓库
3. Render会自动读取 `render.yaml` 配置
4. 在Environment Variables中设置：
   - `DEEPSEEK_API_KEY` = 你的DeepSeek API key
   - `JWT_SECRET` = 自动生成
5. 点击Deploy，等待部署完成
6. 访问 `https://你的服务名.onrender.com`

## Prompt迭代历程

| 轮次 | 问题 | 解决方案 | 结果 |
|------|------|----------|------|
| 1 | AI回复像客服，太正式 | 改人设为"好朋友"，加口语化要求 | 语气改善 |
| 2 | 反复叫用户"姐妹" | 禁止词放到prompt第一行，❌标记 | 不再出现 |
| 3 | 每次都以"啊这"开头 | 删除固定示例短语，改为自然描述 | 开头多样化 |
| 4 | 不管什么问题都接着上文聊 | 加上下文判断规则：换话题就当新话题 | 话题切换自然 |
| 5 | 堆砌星象术语没有中心观点 | 加"回复要有统一方向和观点" | 观点清晰 |
| 6 | AI在回复里标注"(今日运势角度)" | 删除ABCD标签+禁止括号标注 | 输出干净 |
| 7 | 每条回复都差不多长 | 严格长度规则：轻松2-3句/复杂5-6句 | 长短变化 |
| 8 | 反复说"说真的"，缺少情绪 | 禁止"说真的"+加情绪表达规则 | 风格自然有温度 |
